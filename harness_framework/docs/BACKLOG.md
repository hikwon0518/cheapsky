# Backlog

> 이 문서의 한 줄 요지: MVP 범위 밖 아이디어를 **잠시 내려놓는 서랍**. 개발 중 떠오른 "이거 더 하면 좋을 텐데"를 즉시 구현하지 않고 여기에 기록한다 (ADR-016).

## 사용 규칙
- 새 아이디어 = 1~2줄 추가, 개발 계속
- PRD "MVP 제외 사항"과 ADR-026의 Core 범위는 개발 기간 중 **수정 금지**
- 우선순위 매기지 않는다. 지금은 잊어버리기 위한 기록
- MVP 완성 후 정기 리뷰에서 Deferred → 다음 cycle 승격
